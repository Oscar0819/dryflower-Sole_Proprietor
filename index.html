앱 이름을 입력하세요

저작권 및 출처 표기
예: 모든 요약은 원문 링크를 포함하며, 원 저작권자를 표기합니다.
문의 및 신고
문의: contact@your-domain.com / 저작권 신고: abuse@your-domain.com
아래 JSON-LD(구조화 데이터)는 선택사항입니다. 붙여넣으면 검색엔진이 운영자 정보를 이해하는 데 도움이 됩니다.
{
  "@context": "https://schema.org",
  "@type": "NewsMediaOrganization",
  "name": "홍길동미디어",
  "url": "https://your-domain-or-github.io",
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "customer support",
    "email": "contact@your-domain.com"
  }
}
저장 방법: 이 파일을 index.html로 GitHub repo 루트에 올리고 GitHub Pages를 활성화하세요. 또는 docs/ 폴더에 올릴 수도 있습니다.

Play 콘솔에 기입할 때 복사하기 좋은 문구 (예시)

운영 주체: 홍길동(개인사업자)
형태: Individual / Sole Proprietor
사업자등록번호: <사업자등록번호 입력>
공식 웹사이트: https://your-domain-or-github.io
연락 이메일: contact@your-domain.com
콘텐츠 출처: AI 요약 + 원문 링크 제공 (원문 출처를 명시합니다)
저작권: 원저작권자의 권리를 존중하며, 원문 링크 및 출처를 항상 표기합니다.
이 문구를 Play 콘솔의 뉴스 앱 선언란에 붙여넣으면 됩니다.
추가 팁: 웹사이트(또는 GitHub Pages)에 운영자 이름·연락처·사업자번호(가능한 경우)를 명확히 표기해 두세요. 구글 검증 과정에서 필요한 정보입니다.
